# Skyfall-GS (Sparse-view Scene Reconstruction)

## Summary
Skyfall-GS를 치아 도메인에 적용하기 위해 진행했던 sparse-view GS 관련 실험 아카이브. 현재 메인 프로젝트는 별도 폴더인 `projects/dental-texture-pipeline`으로 분리되었고, 이 폴더는 Skyfall-GS 자체와 직접 연결되는 하위 실험을 보관한다.

## Focus
- sparse view 기반 3D Gaussian Splatting 실험 기록
- Skyfall-GS 구조 분석 및 도메인 적응 실험
