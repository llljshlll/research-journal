# Skyfall-GS (Sparse-view Scene Reconstruction)

## Summary
sparse view 환경에서 scene reconstruction을 안정화하기 위해 **Skyfall-GS** 구조를 분석하고 적용 가능성을 탐색 중이다. 
Skyfall-GS가 사용하는 diffusion 모델인 **FLUX**의 구조를 함께 분석한다.

## Focus
- sparse view 기반 3D Gaussian Splatting 최적화
- FLUX ↔ MV-Adapter 구조적 관계 분석
