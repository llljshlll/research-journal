# Skyfall-GS (Sparse-view Scene Reconstruction)

## Summary
MV-adapter로 만든 sparse view 환경에서 3D Gaussian Splatting을 통한 scene reconstruction을 안정화하기 위해 **Skyfall-GS** 구조를 분석하고 적용 가능성을 탐색 중이다. 

## Focus
- sparse view 기반 3D Gaussian Splatting 최적화
