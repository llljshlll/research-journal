# Dental Texture Pipeline

## Summary

정확한 치아/잇몸 mesh를 기반으로, geometry를 유지하면서 realistic하고 multi-view consistent한 texture appearance를 만드는 파이프라인을 정리하는 프로젝트.

## Focus

- mesh-guided dental appearance generation
- multi-view consistency 확보
- relighting 기반 appearance normalization
- 최종 Gaussian Splatting reconstruction
