## 기존 문제

- diffusion이 view마다 다른 albedo / lighting 생성  
→ 동일 geometry에서도 rendering mismatch  

- 결과:
  - NeRF: 얼룩 / 노이즈  
  - GS: blotchy artifact  

- 조명 특성:
  - global consistency 없음  
  - pixel-level perceptual shading  

→ 물리적으로 일관된 lighting 아님  

- 현재 pipeline:
  - noise 생성 → NeRF 정렬 → GS explicit화  
→ noise 누적 + artifact 증폭  

---

## 이후 방향

### 1. Intrinsic Decomposition (핵심)

- 목표:
  - albedo / lighting 분리  
  - lighting inconsistency 제거  

- 적용 위치:

  **(A) Image space**
  - 빠름, 단순  
  - 정렬 고려 없음  

  **(B) NeRF space**
  - 정렬된 상태에서 분리  
  - NeRF artifact 영향 있음  

→ 두 방식 비교 필요  

---

### 2. Gaussian Splatting

- 기존 유지:
  - unconstrained densification  
  - geometry constraint GS  

→ texture 깨진 상태 그대로 학습됨  

---

### 3. Multi-view 생성

- multi-view diffusion  
  → semantic 깨짐  

- single-view 생성  
  → lighting inconsistency  

→ trade-off 존재
