# 2026_05_26_Meeting

## 목표

Semantic mesh 기반 realistic and consistent dental real3D model / texture dataset 생성

---

## 문제 정의

기존 open-source image-to-image / inpainting 모델의 dental domain 성능 부족   

realistic 치아 이미지 생성 뿐 아니라, tooth / gum semantic 정보를 반영하면서, hole filling하는 성능도 부족  

Gemini / Nano Banana는 realistic quality는 높지만, 여러 view를 독립적으로 생성할 경우 multi-view consistency 부족  

---

## 핵심 아이디어

기존 open model의 dental semantic-aware filling 성능 부족

초기 high-quality realistic image 생성을 위한 Gemini / Nano Banana 사용

첫 view 결과를 consistency anchor로 사용

이후 view는 mesh 회전으로 새로 드러나는 hole 영역만 semantic-aware filling

이를 통해 consistent multi-view image dataset 생성

---

## 학습 방향

Gemini / Nano Banana를 이용해 초기 anchor 및 hole filling 결과 생성

생성된 결과를 teacher data로 활용

SD3 기반 ControlNet + inpainting 모델 학습 검토

최종 목표는 Gemini / Nano Banana 의존이 아닌, 저렴하고 제어 가능한 open model 학습

---

## 실험 사항

### 1. Semantic-colored hole mask 실험

flux-fill에서 Binary hole mask 대신 semantic 정보를 포함한 hole mask 사용 가능성 확인  
  
ex) magenta = gum hole / cyan = tooth hole.   
  
=> flux-fill의 mask는 Binary mask만 받음. 그래서 대신, segmentation에 따라 치아/잇몸 따로 생성  
PROMPT_GUMS = (
    "This is a dental/teeth image. "
    "The masked region contains ONLY gum tissue — soft pink gingival tissue surrounding the teeth. "
    "Fill every masked pixel with realistic, natural gum (gingiva) tissue: "
    "consistent pink-red color, smooth soft-tissue texture, "
    "matching the tone and shading of adjacent visible gum areas. "
    "No teeth, no dark voids, no shadow artifacts in this region. "
    "Preserve all geometry outside the mask exactly."
)

PROMPT_TEETH = (
    "This is a dental/teeth image. "
    "The masked region contains ONLY tooth enamel surfaces — crown or exposed tooth structure. "
    "Fill every masked pixel with realistic, natural tooth enamel: "
    "white to off-white color, smooth enamel texture, "
    "matching the brightness and tone of adjacent visible tooth surfaces. "
    "No gums, no dark voids, no shadow artifacts in this region. "
    "Preserve all geometry outside the mask exactly."
)
