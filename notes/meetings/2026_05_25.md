# SDXL ControlNet Segmentation + Inpainting 실험

SDXL에서 segmentation ControlNet과 mask inpainting을 함께 사용했을 때, tooth/gum semantic map이 hole region 보완에 도움이 되는지 확인하였다.  
비교 기준으로는 segmentation 없이 mask 기반 inpainting만 수행한 FLUX Fill 결과를 함께 확인하였다.

---

## 1. 실험 목적

- 입력:
  - 마젠타 hole region이 포함된 pre-render image
  - 마젠타 영역에서 생성한 inpainting mask
  - tooth/gum segmentation map
- 비교:
  - FLUX Fill: segmentation 없이 mask inpainting만 수행
  - SDXL: segmentation ControlNet + mask inpainting 동시 적용

목표는 단순히 마젠타 영역을 제거하는 것뿐 아니라, 치아/잇몸 영역이 semantic map에 맞게 유지되는지 확인하는 것이다.

### 관찰

- FLUX Fill은 전체적인 색감과 질감은 비교적 자연스럽게 보완하지만, segmentation 정보가 없기 때문에 tooth/gum 경계나 작은 hole 영역의 구조를 명시적으로 제어하지 못함
- SDXL은 segmentation map을 함께 입력할 수 있으나, 현재 결과에서는 경계선과 얇은 국소 영역에서 마젠타 잔상 또는 색 번짐이 남는 경우가 있음
- 특히 edge 부분이나 작은 hole region에서는 inpainting이 안정적으로 닫히지 않거나, 주변 색과 부자연스럽게 연결되는 문제가 남음

## 3. Magenta Init 문제

처음에는 input image의 마젠타 영역을 그대로 둔 상태에서 SDXL inpainting을 수행하였다.  
이 경우 SDXL이 마젠타 색을 제거하기보다 결과 이미지에 더 강하게 반영하는 현상이 나타났다.

### 관찰

- 마스크 영역 내부에 마젠타 값이 그대로 남아 있으면, SDXL 결과에서도 마젠타 계열 색이 강하게 생성됨
- 마젠타는 hole 표시용 marker이지만, 모델 입장에서는 이미지의 실제 색/texture 정보로 해석될 수 있음
- 따라서 SDXL에는 마젠타 marker를 그대로 넣는 것보다, inpainting 전 초기 이미지를 한 번 정리한 뒤 입력하는 방식이 더 적합함

---

## 4. Clean Init 적용

마젠타 영역을 그대로 두지 않고, segmentation map을 기준으로 마스크 내부를 치아/잇몸 영역으로 나누어 초기화하였다.  
각 영역은 주변 치아 색과 잇몸 색의 평균값으로 채운 뒤 SDXL inpainting input으로 사용하였다.

### 관찰

- clean init 적용 후에는 마젠타 색이 직접적으로 증폭되는 문제는 줄어듦
- 다만 평균색으로 채운 영역이 넓거나 경계가 복잡한 경우, SDXL이 해당 영역을 자연스러운 geometry로 완전히 복원하지 못함
- 치아와 잇몸의 큰 영역 구분에는 segmentation map이 도움이 될 수 있으나, 치아 사이 경계, 잇몸 라인, 얇은 empty texel 주변에서는 추가 제어가 필요함

---

## 5. 검토 결과

- SDXL ControlNet + Inpainting 조합은 실행 가능하며, segmentation map과 hole mask를 분리 입력하는 실험 경로는 확인됨
- 하지만 현재 pretrained SDXL segmentation ControlNet은 일반 segmentation map 기준이므로, tooth/gum semantic map을 정밀하게 따르는 수준은 제한적임
- 마젠타 marker를 그대로 input으로 넣으면 결과에 마젠타가 강하게 반영될 수 있으므로, clean init 또는 marker 제거 전처리가 필요함
- 현재 결과만 보면 FLUX Fill은 자연스러운 texture 보완에는 강점이 있고, SDXL은 semantic map을 넣을 수 있다는 장점은 있으나 edge/local inpainting 안정성이 부족함

---

## 6. 추가 확인 예정

- clean init 생성 방식을 평균색 fill보다 더 자연스럽게 개선
- 치아 영역과 잇몸 영역을 분리하여 순차 inpainting하는 방식 확인
- segmentation ControlNet strength, guidance scale, denoising strength에 따른 결과 변화 확인
- tooth/gum 전용 segmentation ControlNet을 별도로 학습했을 때 현재 pretrained SDXL 결과보다 개선되는지 확인
